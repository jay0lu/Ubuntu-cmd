# Ubuntu useful cmd 

标签（空格分隔）： ubuntu cmd 

---

* add proxy to pip
```
sudo pip --proxy http://web-proxy.mydomain.com install somepackage
```
*create md5 sum
```
md5sum your-file.tar.gz
```
*DNS server active check
```
$ nslookup XXX

```
